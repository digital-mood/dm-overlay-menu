DM Overlay Menu (WordPress plugin)
-------------------------------------
1) Install from Plugins → Add New → Upload Plugin and choose dm-overlay-menu.zip
2) Activate.
3) Go to Settings → DM Overlay Menu and set:
   - Logo URL (header)
   - Mark URL (dark panel)
   - Tagline
   - Colors (blue/dark/orange)
4) Go to Appearance → Menus → Manage Locations and assign your main menu to “DM Overlay Menu”.
5) To create headings (non-clickable) in the orange panel:
   - Add a Custom Link with URL set to # and add child items; the walker will print the parent label as a section heading.
